# LED Screen Stress Test

Web-based tool for generating tiled stress-test patterns for large LED displays.

## Run locally

1. Open `index.html` in a browser.
2. Use fullscreen mode (press F11 in the browser) for normal operation.

## Run hosted version
https://arildj78.github.io/LEDscreenStressTest/

## Controls

- `P`: Toggle control panel.
- `Tab`: Cycle mouse wheel adjustment mode.
- `I`: Toggle single-tile inspect at the tile currently under the mouse cursor.
- Mouse wheel: Adjust active mode (`Tile Shift` or `Rotation`).
- Mouse side buttons: Cycle wheel mode (if browser/hardware supports it).

## Notes

- The app scales with viewport size and works in fullscreen or windowed mode.
- Geometry and optional physical/pixel inputs update calculation info in real time.
